Privacy Policy

This Privacy Policy describes how SocialMediaAgent ("we", "our", "us") collects, uses, and protects user data when using Facebook and Instagram APIs.

Data Collection

We may collect and process the following data:

  • Facebook/Instagram User ID
  • Access tokens (short-lived or long-lived)
  • Page and account IDs
  • Basic profile information (name, profile picture)

Data Usage

Collected data is used strictly for:

  • Publishing and managing content on behalf of the user
  • Automation of social media actions
  • Performance and analytics related to content

Data Sharing

We do not sell, rent, or share user data with third parties. Data is only used internally for the functionality of the application.

Data Retention

User data is retained only as long as necessary to provide services. Data is deleted when:

  • The user revokes access via Facebook/Instagram
  • The user explicitly requests deletion

Data Deletion

Users can request deletion of their data at any time by visiting:

https://cfiod.github.io/socialmedia-agent-legal/deletion.html

Data Security

We implement reasonable security measures to protect user data. Access is restricted and tokens are handled securely.

User Rights

Users have the right to access, modify, or delete their data at any time.
